# FlexRadioBandDecoder
<br>HF+VHF+UHF band decoder for Flex-6300, Flex-6400, Flex-6500, Flex-6600 (decode 5bit BCD codes from radio)
<br>Core -  STM32F103C8T6 ARM development board 
<br>Many TNX RA0SMS!
